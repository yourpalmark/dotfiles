# tmux shortcuts

Prefix = `ctrl`-`b`

#### Prefix shortcuts

Action | Prefix Shortcut
--- |---
Resize pane | `meta`-`arrow`
Respawn pane | `:respawn-pane -k`

#### Commands

Action | Command
--- | ---
New session | `tmux new -s [session_name]`
Attach session | `tmux a -t [session_name]`
Kill session | `tmux kill-session -t [session_name]`
